#### Статуси перекладу

- офіційний — переклад розміщується на платформі та/або у репозиторії оригіналу
- визнаний — переклад розміщується на платформі перекладача, але автор оригіналу має посилання на цей переклад
- незалежний — переклад розміщується на платформі перекладача, та автор не має посилання на цей переклад
  
_Інші статуси можуть бути додані пізніше._
